## Developer-friendly build notes

Every visual element is categorized as Text, CSS Shape, CSS Gradient, SVG Graphic, Transparent PNG, or Background Image. Cards, shadows, buttons, dividers, gradients, rounded panels, and layout spacing are CSS-based. Icons are standalone SVG files. The official logo is a separate PNG and simplified SVG fallback. The hero and news imagery are separate high-resolution image assets.
